# KFramework

Clean, lightweight, a extensible unity framework

An **official fork** from CosmosEngine: https://github.com/mr-kelly/CosmosEngine


# 特点/Features

* 规范的资源管理器，支持高清版、低清版不同资源
* 良好异步风格AssetBundle加载
* AssetBundle打包框架
* 基于约定的、无配置式的UI框架
